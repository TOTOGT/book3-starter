## dm3 Mathematics and Physics Research Lab — Classroom

Welcome to the `book3-starter` template repository for the **G6LLC** classroom on GitHub.  
This repository is the starting point for assignments and projects in the dm3 mathematics and physics research lab.

---

## Getting Started

1. **Accept the assignment** via the GitHub Classroom link provided by your instructor.
2. **Clone your personal copy** of this repository:
   ```bash
   git clone https://github.com/G6LLC/<your-repo-name>
   cd <your-repo-name>
   ```
3. Follow the instructions in each assignment folder under `assignments/`.
